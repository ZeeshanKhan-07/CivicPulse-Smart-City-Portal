üèôÔ∏è CivicPulse ‚Äì Smart City Grievance & Feedback Management Portal
Project Overview
CivicPulse is a citizen-centric Smart City web portal designed for real-time grievance reporting, issue tracking, and civic feedback collection. It acts as a digital bridge between citizens and municipal authorities to ensure quick and efficient resolution of civic issues.

Key Features
Citizen-Centric Grievance Submission: Users can submit complaints (sanitation, street lights, water, roads, etc.) with images and location data.

Role-Based Access: Separate panels for Citizen, Admin, and Department Officer.

Real-time Tracking: Citizens can track the status of their complaints (Pending/In Progress/Resolved).

Grievance Management (Admin): Centralized dashboard for administrators to view, assign issues to officers, and set deadlines/priorities (SLAs).

Officer Workflow: Department Officers can view assigned issues, mark progress, and upload resolution notes and images.

Feedback & Rating: Citizens can rate resolved issues and have the option to reopen if unsatisfied.

Analytics & Reporting: Dashboards with charts showing complaint distribution (category/zone) and SLA performance tracking.

Frontend:	React Js, Tailwind CSS, G-SAP
Backend:	Spring Boot, Spring Security
Database:	MySQL with JPA/Hibernate
Charts/Analytics:	Chart.js / ApexCharts / Recharts
DevOps & APIs:	Postman, GitHub
